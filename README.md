# Reproducible Quant Studies

Python code and notes for the studies published on the Repro Quant site.

This repository is for research scripts, not trading signals. Each study should keep the rule, data source, assumptions, output files, and chart generation steps close together.

## Layout

```text
reproducible-quant-studies/
  studies/
    README.md
    first-study-slug/
      files will be added after the first study is reproduced
  shared/
    reusable helpers can go here later
```

## Disclaimer

These scripts are for learning and reference only. They are not financial advice, trading signals, or recommendations to buy or sell any security.
